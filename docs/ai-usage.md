# 11. Uso de inteligencia artificial (IA)

## Herramientas utilizadas
Durante el desarrollo del proyecto se ha utilizado ChatGPT como herramienta de inteligencia artificial generativa para apoyar el diseño, implementación y documentación del sistema.

---

## Finalidad del uso de IA

La IA se ha utilizado en las siguientes fases del proyecto:

- Diseño inicial de la arquitectura del sistema de alarmas.
- Generación de clases base en Java (Alarm y AlarmManager).
- Propuesta e implementación de funcionalidades avanzadas:
  - Modo circadiano
  - Modo vacaciones
  - Snooze
- Asistencia en la estructura del flujo de trabajo con Git y GitHub.
- Creación de documentación técnica en formato Markdown.
- Elaboración de casos de uso y reflexiones técnicas.

---

## Ejemplos de prompts utilizados

Algunos ejemplos reales de prompts empleados:

- “Diseña un sistema de alarmas inteligente en Java sin interfaz gráfica”
- “Añade modo circadiano con incremento progresivo de volumen”
- “Cómo estructurar ramas en Git para un proyecto académico”
- “Redacta casos de uso en formato tabla para un sistema de alarmas”
- “Añade modo vacaciones a un gestor de alarmas en Java”

---

## Código generado o asistido por IA

La IA contribuyó total o parcialmente en:

- Estructura inicial de clases Java
- Métodos de gestión de alarmas
- Lógica del modo circadiano
- Modo vacaciones
- Ejemplos de Main para pruebas

Todo el código generado fue revisado, adaptado y corregido manualmente para ajustarse a los requisitos del proyecto.

---

## Validación del código

El código generado por IA fue validado mediante:

- Revisión manual de la lógica.
- Ejecución de pruebas en el método `Main`.
- Corrección de errores de diseño y coherencia.
- Adaptación a las restricciones del proyecto (sin interfaz gráfica y estructura simple).

---

## Limitaciones detectadas

Durante el uso de IA se detectaron algunas limitaciones:

- Propuestas demasiado genéricas en algunos casos.
- Necesidad de adaptar el código a un nivel académico concreto.
- Falta de conocimiento del contexto completo del proyecto en algunas respuestas.

---

## Aprendizajes obtenidos

El uso de IA ha permitido:

- Acelerar el desarrollo del proyecto.
- Explorar distintas soluciones de diseño rápidamente.
- Comprender mejor la estructura de aplicaciones orientadas a objetos.
- Mejorar la capacidad de análisis y depuración del código generado.

---

## Conclusión

La inteligencia artificial se ha utilizado como herramienta de apoyo al desarrollo, pero no como sustituto del proceso de diseño y programación.

El resultado final es producto de la combinación entre asistencia de IA y validación manual del desarrollador.
